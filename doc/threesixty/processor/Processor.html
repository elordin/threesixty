threesixty.processor

Processor

Related Doc: package processor

class Processor extends ProcessingMixins with UsageInfo

Holds a list of available processing methods and some meta information.

Most notably knows and executes all conversions from JSON to threesixty.processor.ProcessingStep as well as deductions when the processing strategy is not defined.
